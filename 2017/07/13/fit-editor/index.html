ueditor自定义弹出框

发表于 2017-07-13 | 分类于 javascript

最近公司产品上线了商城模块，为了方便推广，需要在文章里添加一个商品的信息，点击跳转都对应的商品详情页面，文章的编辑器使用的是百度的ueditor，看了下文档，可以使用自定义弹出框来实现这个效果。

第一步

找到ueditor.config.js文件中的toolbars的数组，增加一个mydialog,然后找到labelMap对象，添加'mydialog':'测试',如果不需要全局也可以对单独文件添加

第二步

找到dialogs文件，新建mydialog文件夹，然后在文件夹里新建mydialog.html文件，这个文件自定义页面的代码，任何编辑修改都在这里，如参考ling/link.html文件。

第三步

找到ueditor.all.js文件，搜索btnCmds,在数组里添加mydialog。

搜索iframeUrlMap,把第二步中的的文件路径加进去。

搜索dialogBtns,在ok属性中添加mydialog。
